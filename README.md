# 完整使用说明/GitHub项目
#### 模块成功Hook后会在设置里显示指纹设置选项，在里面打开总开关并设置好密码即可使用。
https://github.com/ghhccghk/mhspay

## 输入的密码会不会被窃取？
#### 密码使用AES加密，在有tee的设备会在tee加解密来保证安全，所有和密码有关的界面做了防录屏截图处理，防止第三方APP截取密码。本模块不会保留密码，密码只能在开启Debug编译的情况下在系统日志里会出现。
